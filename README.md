# 🚚 Blinkit Power BI Dashboard

A visually interactive Power BI dashboard analyzing Blinkit's key metrics like total sales, item distribution, outlet performance, and regional sales insights.

## 📊 Key Highlights

- 💰 **Total Sales:** $1.20M
- 🛒 **Number of Items Sold:** 8523
- ⭐ **Average Rating:** 3.9
- 🧾 Outlet-based filters by:
  - Location Type (Tier 1, Tier 2, Tier 3)
  - Outlet Size (Small, Medium, High)
  - Item Type & Fat Content

## 🖼️ Dashboard Preview

![Blinkit Dashboard](blinkit.preview.png)

## 🧠 Insights Derived

- Tier 3 outlets have the highest sales ($472K+).
- Regular items contribute more than Low Fat items.
- Supermarket Type 1 dominates in sales and item count.
- Trends show peaks around 2018 followed by slight decline.
